        <h1 class="entry-title" itemprop="headline">Kissa tarvitsee kodin koko elämänsä ajaksi</h1>      </header>
          <div class="info-row">
    <div class="single-author">Teksti: Anna Vasala</div>    <div class="single-pic-rights">Kuva: Anna Vasala</div>    <div class="single-date">Uutinen julkaistu: 28.07.2022</div>
    </div>
    <div class="single-excerpt"><p>Kissa on Suomen yleisin lemmikki. Monet ihmiset ihastuvat suloisiin kissanpentuihin. He voivat ajatella, että kissa pärjää aikuisena luonnossa. Jokainen kissa tarvitsee kuitenkin pysyvän kodin ihmisen kanssa.</p>
</div><div class="featured-image  page-header-image-single ">
				<div class="wp-caption thumb-caption rs_skip"><img width="975" height="600" src="https://selkosanomat.fi/wp-content/uploads/2022/07/arkisto_kissat_netti.jpg" class="attachment-full size-full" alt="Hylätty kissa. Suomessa hylätään paljon kissoja joka vuosi." itemprop="image" decoding="async" fetchpriority="high" srcset="https://selkosanomat.fi/wp-content/uploads/2022/07/arkisto_kissat_netti.jpg 975w, https://selkosanomat.fi/wp-content/uploads/2022/07/arkisto_kissat_netti-512x315.jpg 512w, https://selkosanomat.fi/wp-content/uploads/2022/07/arkisto_kissat_netti-768x473.jpg 768w" sizes="(max-width: 975px) 100vw, 975px" /><p class="wp-caption-text thumb-caption-text rs_skip">Vapaana liikkuva kissa etsii suojaa rakennusjätteiden keskeltä.</p></div>
			</div>
    <div class="entry-content" itemprop="text">
      <p>Kissaa ei saa ottaa ennen kuin on miettinyt asiaa huolellisesti.</p>
<p>Kissa voi elää yli kaksikymmentä vuotta. Se tarvitsee koko elämänsä ajaksi pysyvän kodin ja hyvää hoitoa. Kissa tarvitsee kodissaan säännöllisesti ruokaa, rakkautta ja turvallisen ympäristön.</p>
<p>Ihmiset voivat ajatella, että kissa pärjää luonnossa talven yli. Se ei ole totta. Kissa ei kuulu Suomen luontoon.</p>
<p><strong>Kesäkissaa ei pidä ottaa</strong></p>
<p>Suomen luonnossa liikkuu monia tuhansia kissoja, joilla ei ole kotia. Tällaisten kissojen määrä on valtava.</p>
<p>Jotkut ihmiset ottavat kissan lemmikiksi vain kesän ajaksi, esimerkiksi kesämökille. Sellaista kissaa sanotaan kesäkissaksi. Kesäkissat ovat iso ongelma, sillä syksyllä mökille jätetty kissa selviää talvesta huonosti.</p>
<p>Kissat voivat saada talvella pahoja paleltumia. Monet taudit kiusaavat vapaina liikkuvia kissoja.</p>
<p>Hylätyt kissat eivät löydä luonnosta tarpeeksi ruokaa. Ne ovat nälkäisiä ja laihtuvat pahasti. Moni kissa kuolee talven aikana joko nälkään tai johonkin sairauteen. Se on turhaa ja surullista.</p>
<p><strong>Kissa tarvitsee hyvän kodin</strong></p>
<p>Keväällä ja kesällä syntyy paljon uusia kissanpentuja. Ne kaikki ansaitsevat hyvän kodin koko elämänsä ajaksi.</p>
<p>Kissan pitää olla neljätoista viikkoa vanha ennen kuin sen voi ottaa uuteen kotiin. Siihen mennessä pentu on ehtinyt kasvaa tarpeeksi eikä se tarvitse enää emon eli äitinsä maitoa. Kissanpentu on myös saanut sille kuuluvat matolääkitykset.</p>
<p>Emo opettaa pennulle monia kissan elämään kuuluvia taitoja. Silloin kissanpentu selviää itsenäisesti uudessa kodissaan. Tarpeeksi vanhana pikkukissa osaa esimerkiksi käydä kissanvessassa eli hiekkalaatikolla. Pentu osaa myös puhdistaa itsensä.</p>
<p><strong>Omistaja on vastuussa</strong></p>
<p>Suomen laki sanoo, että kissat eivät saa ulkoilla ilman valvontaa taajamassa eli tiheään asutulla alueella. Vapaana liikkuva kissa voi jäädä auton alle tai loukkaantua muulla tavalla.</p>
<p>Jotkut ihmiset voivat jopa kiusata tai vahingoittaa vapaana liikkuvaa kissaa. Lisäksi kissa voi syödä lintuja tai vaikka oravanpoikasia.</p>
<p>Kissan omistaja on aina vastuussa lemmikistään.</p>
<p><strong>Kissa kannattaa leikkauttaa</strong></p>
<p>Kissa kannattaa leikkauttaa, ettei se tee lisää pentuja.</p>
<p>Jokainen kissa kannattaa leikkauttaa. Leikkauksella tarkoitetaan eläinlääkärin tekemää toimenpidettä, jossa kissa tehdään lisääntymiskyvyttömäksi. Se tarkoittaa, ettei kissa voi enää saada pentuja.</p>
<p>Leikkaamaton kissa saa luonnossa lisää pentuja, jotka tekevät uusia pentuja aikuisina. Silloin ongelma vain kasvaa ja kasvaa.</p>
<p><strong>Turvasiru auttaa löytämään omistajan</strong></p>
<p>Kissalle kannattaa laittaa turvasiru.</p>
<p>Kissan omistaja voi huolestua kovasti, jos kissa karkaa. Turvasiru auttaa löytämään oikean omistajan, jos joku toinen ihminen saa kissan kiinni.</p>
<p>Turvasiru on pieni mikrosiru, joka laitetaan kissan nahan alle. Turvasiru ei häiritse kissaa tai aiheuta sille kipua. Sirun avulla saadaan selville omistajan tiedot. Näin karannut kissa pääsee takaisin kotiin.</p>
<p><strong>Kissa tarvitsee rokotuksia</strong></p>
<p>Kissan terveyttä pitää tarkkailla säännöllisesti.</p>
<p>Lemmikiksi otettu kissa pysyy hyvässä kunnossa, kun sitä hoidetaan oikein. Sen terveyttä tarkkaillaan säännöllisesti. Kissa viedään eläinlääkäriin, jossa se loukkaantuu tai se vaikuttaa sairaalta.</p>
<p>Kissa on myös hyvä rokottaa joitakin tauteja vastaan. Rokotus tarkoittaa pistosta, jonka tarkoitus on ehkäistä kissaa sairastumasta johonkin tiettyyn sairauteen. Rokotusten avulla ehkäistään muun muassa kissaflunssaa.</p>
<p>Vapaasti liikkuva kissa voi saada ulkoa loisia ja tauteja. Ne saattavat tarttua myös ihmisiin. Tämänkin takia kissa on parempi pitää sisällä tai ulkoiluttaa sitä vaikka valjaissa.</p>
<p>Kissa voi hyvin ja elää onnellista elämää, kun sillä on koti ympäri vuoden. Kotona pysyvä kissa ei aiheuta turhaa huolta itselle eikä naapureille. Silloin myös kissan omistajalla on hyvä omatunto. Hän osaa nauttia lemmikkinsä seurasta.</p>
<p><em>Juttu on julkaistu Selkosanomissa heinäkuussa vuonna 2020.</em></p>

// Element to move, time in ms to animate
function scrollTo(element, duration) {
  var e = document.documentElement;
  if (e.scrollTop === 0) {
    var t = e.scrollTop;
    ++e.scrollTop;
    e = t + 1 === e.scrollTop-- ? e : document.body;
  }
  scrollToC(e, e.scrollTop, element, duration);
}

function scrollToC(element, from, to, duration) {
  if (duration <= 0) {
    return;
  }
  if (typeof from === "object") {
    from = from.offsetTop;
  }
  if (typeof to === "object") {
    to = to.offsetTop;
  }
  scrollToX(element, from, to, 0, 1 / duration, 20, easeOutCuaic);
}

function scrollToX(element, xFrom, xTo, t01, speed, step, motion) {
  if (t01 < 0 || t01 > 1 || speed <= 0) {
    element.scrollTop = xTo;
    return;
  }
  element.scrollTop = xFrom - (xFrom - xTo) * motion(t01);
  t01 += speed * step;

  setTimeout(function () {
    scrollToX(element, xFrom, xTo, t01, speed, step, motion);
  }, step);
}

function easeOutCuaic(t) {
  t--;
  return t * t * t + 1;
}
